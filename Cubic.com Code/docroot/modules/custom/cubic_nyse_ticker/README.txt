How to configure this module?
--------------------------------

Add the following code to the settings.php file

/**
 * Settings for the cubic_nyse_ticker module
 */
$settings['cubic_nyse_ticker.function'] = 'GLOBAL_QUOTE';
$settings['cubic_nyse_ticker.symbol'] = 'NYSE:CUB';
$settings['cubic_nyse_ticker.apikey'] = 'DSMO4Q94TOQK1D7W';
